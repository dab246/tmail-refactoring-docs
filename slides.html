Pillar 2 — AppEventBus

AppEventBus: Type-safe, Zero Coupling

class AppEventBus extends GetxService {
  final _ctrl =
    StreamController<AppEvent>.broadcast();

  Stream<AppEvent> get stream => _ctrl.stream;

  void dispatch(AppEvent event) {
    if (!_ctrl.isClosed) _ctrl.add(event);
  }

  @override
  void onClose() {
    _ctrl.close();     // cleanup khi logout
    super.onClose();
  }
}

// Sealed class — compile-time exhaustive
sealed class AppEvent {}

final class MailboxSelectedEvent
    extends AppEvent {
  final PresentationMailbox mailbox;
  MailboxSelectedEvent(this.mailbox);
}

final class EmailDeletedEvent
    extends AppEvent {
  final List<EmailId> emailIds;
  EmailDeletedEvent(this.emailIds);
}
// Subscriber — chỉ nhận type mình cần
class ThreadController
    extends BaseController
    with EventBusSubscriberMixin {

  @override
  void onInit() {
    super.onInit();
    // Chỉ nhận MailboxSelectedEvent
    _eventBus.stream
      .whereType<MailboxSelectedEvent>()
      .listen(_onMailboxSelected)
      .addTo(compositeSubscription);
      // ↑ auto cancel khi dispose
  }
}
LỢI ÍCH
  • Zero coupling — controllers không biết nhau
  • Type-safe tại compile time
  • Thêm event mới → không sửa code cũ (OCP)
  • Auto cleanup — không leak
  • Dễ test: inject mock stream

Nguyên tắc

8 Quy tắc áp dụng toàn codebase

✅ DOs

  • Constructor injection only — không Get.find() trong business logic
  • Mỗi controller ≤ 300 lines
  • Mỗi constructor ≤ 8 dependencies
  • Mỗi controller ≤ 10 Rx fields
  • EventBus chỉ cho cross-feature events
  • Subscribe bằng whereType<T>()
  • Auto cleanup: .addTo(compositeSubscription)
  • Domain Contracts — không import implementation

❌ DON'Ts

  • Không còn Rxn<UIAction> observables
  • Không còn Get.find<T>() trong field declarations
  • Không dispatch event trong handler cùng loại (loop)
  • Không subscribe toàn bộ stream rồi if-else
  • Không tạo GetxService chứa tất cả state (God Service)
  • Không dùng EventBus cho UI events (button tap, scroll)
  • Không cast event thủ công (as MyEvent)
  • Không giữ StreamSubscription mà không cancel
